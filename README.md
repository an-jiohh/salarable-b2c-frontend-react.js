# b2c-frontend-react.js
Salarable b2c 부분 서비스를 위한 react repository

## repository 개요
salarable의 b2c 부분 서비스를 위한 react기반 프론트엔드 repository입니다.
ml-backend-server과 연동하여 모의면접 관련 서비스를 제공합니다.

## 기술 스택
- React
- Vite
- tailwind
- Shadcn

## 주요 기능 및 컴포넌트
- 모의면접 사전 세팅
  - 포트폴리오 입력 및 마스킹 컴포넌트(모달)
- 모의면접 연결 및 진행
  - 대답 컴포넌트
  - 대기 컴포넌트
  - 종료 컴포넌트
  - 종료 대기 컴포넌트
  - 세팅 컴포넌트
  - 장비 연결 컴포넌트
- 로그인
